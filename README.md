# ncx2xhtml

Converter from EPUB 2 TOC (ncx) to EPUB 3 TOC (xhtml).

Output is now readable (pretty printing output XML)

# Usage

Type into Terminal:

	python3 ncx2xhtml.py toc.ncx nav.xhtml
